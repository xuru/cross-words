Welcome to cross-words' documentation!
======================================

.. note::

    This package is in beta version. It is fully working but will likely get more features in the future.
    See `the repo on GitHub <https://github.com/data-chirps/cross-words>`_ for the latest updates.

cross-words is a python module that allows you to easily create a corpus of documents with parameterized entities.

The main goal of cross-words is to offer an easy way to create either sentences or stories for use in chat bot training. As of May 2018, it is mostly designed to be used with Rasa NLU/Core

This mini-website serves as a reference documentation for interested users.
