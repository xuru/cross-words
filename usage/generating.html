2.2. Generating training files with cross-words¶

cross-words mainly comes with 2 functions :

  • parse_input
  • generate

These are automatically imported with

from xwords import *

All other functions are implementation details.

xwords.parse_input(input_path)[source]¶

Parsing input config file and creating list of paragraphs (lists of lines)

input_path:
path to config file
List
List (one element for each paragraph) of lists (one element for each line within the paragraph)
xwords.generate(input_path, output_path='./xwords/outputs/', training_ratio=1.0, n_sub=None, for_story=False)[source]¶

Generates train and test files for Rasa NLU/Core from config file

input_path:
path to config file
training_ratio:
percentage of sentences/conversations to be kept separate in a test set
output_path:
path to the desired location for generated files
None. Writes the number of sentences in the created training and testing sets files
